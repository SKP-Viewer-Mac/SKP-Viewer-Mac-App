## Overview  

SKP Viewer for macOS is a fast and reliable application designed to open, inspect, and present SketchUp (.skp) models without needing the full SketchUp software. Built for architects, designers, and engineers, it provides an efficient way to visualize 3D projects directly on Mac systems.  

This viewer offers full navigation control — rotate, pan, and zoom with precision to explore complex structures and detailed geometry. It supports real-time rendering, making it easy to analyze lighting, materials, and textures while maintaining smooth performance on both Intel and Apple Silicon Macs.  

The interface is clean and focused on usability, allowing professionals and students alike to review 3D content without distractions. SKP Viewer also provides a presentation mode for showcasing designs to clients or collaborators.  

With quick file loading, accurate model scaling, and broad format support, SKP Viewer stands as an essential companion for anyone working with SketchUp models on macOS.
